Informations

Projet template de librairie avec un projet Qmake (fichier .pro pour IDE QtCreator)

prérequis

  • Windows:
    • Microsoft Visual Studio 2013
  • Mac :
    • Xcode
  • Cross Plaform
    • QtCreator
    • kit de developpement Qt associé à l'environnement de développement (ex MSVC2013 64bits Opengl pour Windows)
  • Variable d'environnement BCOMDEVROOT définie sur un dossier (ex c: pour Windows, /usr/lib/ pour mac)

Compilation et installation

  • Ouvrir le projet TemplateLib.pro avec QtCreator

Ajout phase d'installation

Aller dans projets (icone à gauche), cliquer sur Ajouter l'étape Compiler dans Etapes de compilation, choisir Make, et saisir install dans Arguments de Make

Compilation

  • clic droit sur le projet-> *éxécuter qmake* (pour bien charger la configuration du projet, a faire a chaque changement de configuration du projet)
  • clic droit sur le projet-> tout recompiler

Le projet est compilé puis est installé dans le dossier *BCOMDEVROOT%/bcomBuild/TemplateLib* (interfaces et lib)
